JS Beautify reformats and re-indents JavaScript, CSS, and HTML. This guide walks you through installing it and running your first beautification.

## Prerequisites

You need one of the following runtimes:

- **Node.js 14 or later** — for the npm package and CLI
- **Python 3** — for the `jsbeautifier` and `cssbeautifier` pip packages

<Tip>
  You can also use JS Beautify in the browser without any installation. See [Web browser / CDN usage](/installation/web-browser).
</Tip>

## Steps

<Steps>
  <Step title="Install js-beautify">
    Choose your runtime and install the package.

    <Tabs>
      <Tab title="Node.js (global CLI)">
        ```bash
        npm install -g js-beautify
        ```

        This installs three executables: `js-beautify`, `css-beautify`, and `html-beautify`.
      </Tab>
      <Tab title="Node.js (local library)">
        ```bash
        npm install js-beautify
        ```

        Use this when you want to call the beautifier programmatically from your own Node.js code.
      </Tab>
      <Tab title="Python">
        ```bash
        pip install jsbeautifier
        ```

        To also beautify CSS from Python, install the companion package:

        ```bash
        pip install cssbeautifier
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Beautify a JavaScript file">
    <Tabs>
      <Tab title="Node.js CLI">
        ```bash
        js-beautify foo.js
        ```

        Beautified output is written to stdout. To rewrite the file in place, add `--replace`:

        ```bash
        js-beautify --replace foo.js
        ```
      </Tab>
      <Tab title="Node.js library">
        ```javascript
        const beautify = require('js-beautify');

        const ugly = 'function greet(name){return "Hello, "+name;}';
        const pretty = beautify.js(ugly, { indent_size: 2 });
        console.log(pretty);
        // function greet(name) {
        //   return "Hello, " + name;
        // }
        ```
      </Tab>
      <Tab title="Python">
        ```python
        import jsbeautifier

        ugly = 'function greet(name){return "Hello, "+name;}'
        pretty = jsbeautifier.beautify(ugly)
        print(pretty)
        # function greet(name) {
        #     return "Hello, " + name;
        # }
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Beautify a CSS file">
    <Tabs>
      <Tab title="Node.js CLI">
        ```bash
        css-beautify styles.css
        ```
      </Tab>
      <Tab title="Node.js library">
        ```javascript
        const beautify = require('js-beautify');

        const ugly = 'body{margin:0;padding:0;}.container{display:flex;}';
        const pretty = beautify.css(ugly, { indent_size: 2 });
        console.log(pretty);
        // body {
        //   margin: 0;
        //   padding: 0;
        // }
        // .container {
        //   display: flex;
        // }
        ```
      </Tab>
      <Tab title="Python">
        ```python
        import cssbeautifier

        ugly = 'body{margin:0;padding:0;}.container{display:flex;}'
        pretty = cssbeautifier.beautify(ugly)
        print(pretty)
        # body {
        #     margin: 0;
        #     padding: 0;
        # }
        # .container {
        #     display: flex;
        # }
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Beautify an HTML file">
    <Tabs>
      <Tab title="Node.js CLI">
        ```bash
        html-beautify index.html
        ```
      </Tab>
      <Tab title="Node.js library">
        ```javascript
        const beautify = require('js-beautify');

        const ugly = '<html><head><title>Hi</title></head><body><p>Hello</p></body></html>';
        const pretty = beautify.html(ugly, { indent_size: 2 });
        console.log(pretty);
        // <html>
        //   <head>
        //     <title>Hi</title>
        //   </head>
        //   <body>
        //     <p>Hello</p>
        //   </body>
        // </html>
        ```
      </Tab>
    </Tabs>

    <Note>
      The Python `jsbeautifier` package only handles JavaScript. Use the Node.js package or the browser build to beautify HTML.
    </Note>
  </Step>
</Steps>
